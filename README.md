# Welcome to the LLM-Shortcuts package!
This package is intended to provide users with a set of quick tools to work with local LLMs using LM studio. 

In particular, it supports:

1) Using single-run queries with simple LLMs, multimodal models, and tokenisers (see the RunLLM folder)
2) An interactive chatbot directly in the console of your IDE, based on the model of your choice
3) A set of RAG routines to prepare, vectorise, and search for relevant data in a vectorised database

I hope that this package will help its users to get a bit closer to using large language models, providing a simple point of entry to people not technically nested in AI.
